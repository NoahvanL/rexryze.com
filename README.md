# RexRyze Website

Professionele website voor RexRyze – websites bouwen en verkopen aan lokale bedrijven.

## Bestanden

- `index.html` – Homepagina
- `diensten.html` – Dienstenpagina
- `portfolio.html` – Portfolio met 4 demo-websites
- `prijzen.html` – Prijzenpagina met interactieve calculator
- `contact.html` – Contactpagina met werkend formulier
- `demo-restaurant.html` – Demo: Restaurant "De Gouden Lepel"
- `demo-garage.html` – Demo: Autogarage "Van Dijk Auto Service"
- `demo-kapper.html` – Demo: Kapper "Studio Snip"
- `demo-loodgieter.html` – Demo: Loodgietersbedrijf "Pro Loodgieter"

## Hoe uploaden naar GitHub + domein (rexryze.com)

1. Maak een nieuw GitHub repository aan (bijv. `rexryze-website`)
2. Upload alle bestanden naar de root van de repo (of gebruik deze map)
3. Ga naar Settings → Pages
4. Source: "Deploy from a branch" → Branch: `main` → Folder: `/ (root)`
5. Wacht 1-2 minuten → je site is live op `https://jouwusername.github.io/rexryze-website`
6. Koppel je eigen domein rexryze.com via GitHub Pages custom domain (of via je domeinregistrar CNAME/A record naar de GitHub Pages)
